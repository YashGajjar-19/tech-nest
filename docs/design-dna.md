# 🧬 Tech Nest Design DNA

Design DNA = **rules every component obeys**.

If you follow this, your UI will automatically look professional.

---

# 1️⃣ Brand Personality (Most Important)

Before colors or components:

**What does Tech Nest feel like?**

### Tech Nest Personality

```
Calm Intelligence
Minimal Authority
Precision Technology
Trustworthy Decisions
```

NOT:

* flashy
* gaming UI
* neon tech
* startup hype

Think:

👉 Apple intelligence
👉 Linear clarity
👉 Vercel engineering precision

---

# 2️⃣ Color Philosophy — 60 / 30 / 10

You already chose correctly. Now we lock behavior.

## 60% — Foundation

Used ONLY for:

* page background
* layout
* whitespace

```
Light → #FAFAFA
Dark → #000000
```

Rule:
👉 Background must disappear.

---

## 30% — Surfaces

Used for:

* cards
* navbar
* panels
* inputs

```
Light → #FFFFFF
Dark → #0A0A0A
```

Rule:
👉 Surfaces create depth, not shadows.

---

## 10% — Accent (Sacred Color)

```
Indigo = Intelligence
```

Used ONLY for:

* active navigation
* focus state
* primary CTA
* selection
* links

Never decorate with accent color.

Accent = **meaning**, not decoration.

---

# 3️⃣ Typography DNA

Most products fail here.

## Font Stack

You already chose correctly:

```
Geist Sans → Primary
Geist Mono → Data / Specs
```

---

## Text Hierarchy

### Hero Heading

```
font-medium
tracking-[-0.02em]
leading-[1.1]
```

NOT bold.

Bold = marketing website.

Medium weight = premium product.

---

### Section Heading

```
text-xl
font-medium
```

---

### Body

```
text-sm
text-text-secondary
```

---

### Data / Specs (VERY IMPORTANT for Tech Nest)

Phones = data-heavy.

Use mono font:

```
font-mono text-sm
```

This alone will make Tech Nest feel unique vs GSMArena.

---

# 4️⃣ Spacing System (Hidden Superpower)

You must follow ONE rule:

## **8px Grid System**

Allowed spacing:

```
2  → 8px
4  → 16px
6  → 24px
8  → 32px
12 → 48px
16 → 64px
24 → 96px
```

Never random spacing.

Consistency = premium feel.

---

# 5️⃣ Radius System (VERY IMPORTANT)

Cheap UI = random border radius.

Tech Nest radius language:

```
Buttons → rounded-lg
Cards → rounded-xl
Dialogs → rounded-2xl
Full pills → avoid
```

No exaggerated rounding.

Linear/Vercel rule:
👉 **confidence = controlled curves**

---

# 6️⃣ Shadow Philosophy

Most developers destroy design here.

## Tech Nest Rule:

**Shadows are almost invisible.**

Light mode:

```
shadow-sm only
```

Dark mode:

```
NO shadows
Use contrast instead
```

Depth comes from surfaces — not shadows.

---

# 7️⃣ Motion DNA (This is HUGE)

Your product must feel alive but calm.

### Motion Style

```
Slow start
Fast settle
No bounce
No gimmicks
```

---

## Standard Animation

Always use:

```ts
transition:
cubic-bezier(0.16, 1, 0.3, 1)
```

(Vercel + Apple curve)

---

### Animation Timing

| Action          | Duration |
| --------------- | -------- |
| Hover           | 150ms    |
| Open menu       | 220ms    |
| Page transition | 300ms    |
| Modal           | 260ms    |

Consistency builds subconscious trust.

---

# 8️⃣ Interaction Language

Every interaction must answer:

> “Does this help decision making?”

Examples:

✅ subtle hover
✅ gentle scale 1.02
✅ soft opacity change

Avoid:

❌ glow
❌ neon effects
❌ aggressive motion

---

# 9️⃣ Component Philosophy

Every component must follow:

```
Quiet by default
Clear on hover
Confident when active
```

---

### Buttons

Primary:

```
solid + accent
```

Secondary:

```
ghost / minimal
```

Never multiple loud buttons.

---

### Cards

Rules:

* No borders + heavy shadows together
* Prefer subtle border
* Clean spacing
* Information first

---

# 🔟 Layout DNA

Your layouts should feel like:

```
Apple Product Page
+
Linear Dashboard
+
Notion Clarity
```

Meaning:

* wide breathing space
* centered focus
* low visual noise

---

# 🧠 The Golden Rule of Tech Nest

Every screen must answer:

> **“Can a user make a better tech decision faster?”**

Design supports thinking.
